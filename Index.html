Inicio



El diseño y desarrollo de un sitio web, conlleva a un planteamiento cuyas etapas se deben definir para poder establecer unas expectativas en cuanto a objetivos a cumplir, plazos de tiempo de realización y presupuesto.

El desarrollo de esta página web se llevó a cabo mediante el planteamiento que tuvo la fase anterior, en donde se implementó el mockup; ésta es una etapa importante para la realización del trabajo, ya que podemos apreciar un bosquejo de cómo deberá quedar nuestra página.

Es necesario y fundamental la implementación de un lenguaje de programación.

El lenguaje que se utilizó para este proyecto fue: HTML, este lenguaje se encarga básicamente de la estructura de la página; del mismo modo, y de la mano del HTML está el CSS, que se encarga principalmente de darle vida a la misma, ya que sin éste la página web sería simplemente un montón de letras, resultando poco llamativa.
